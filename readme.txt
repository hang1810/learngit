本地项目推送git
1、git init 把这个文件夹变成Git可管理的仓库
2、使用命令 git add . 把项目添加到暂存区里面去，不要忘记后面的小数点“.”，意为添加文件夹下的所有文件
3、用 git commit -m "第一次提交"把项目提交到仓库。
4、在Github上创建好Git仓库后通过命令git remote add origin git@github.com:xxxx/xxx.git和本地仓库进行关联
